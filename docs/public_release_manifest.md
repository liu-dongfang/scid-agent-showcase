# Public Release Manifest

## Retained In This Showcase Repo

- Core orchestration and controller logic
- Event bus and session state data structures
- Schema models, registry, and JSON schema export
- Structured logging utilities
- Replay/import helpers and compact report reconstruction
- Synthetic workflow definitions and toy question manifests
- Minimal tests, README, environment template, and architecture documentation

## Removed Or Intentionally Excluded

- All real or sensitive data
- `.env` and any local/private configuration
- Paper experiment outputs, ablations, and unpublished evaluation assets
- Prompt libraries, notebook analysis, and not-yet-curated resources
- Human evaluation outputs, large logs, runtime artifacts, and deployment leftovers
- Frontend build outputs and unrelated developer-local files

## Should Remain Private In The Research Repo

- Full prompt and evaluator resources
- Real interview assets, any non-synthetic transcripts, and internal annotations
- Full question banks and paper-specific study materials not ready for release
- Experiment tracking, benchmark suites, and unpublished error analyses
- Internal deployment scripts, operational credentials, and private environment notes
