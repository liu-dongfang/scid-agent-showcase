# Contributing

Thanks for your interest in the public showcase repository.

## Scope

This repository is intentionally smaller than the private research codebase. Contributions are most helpful when they improve the public showcase itself, for example:

- documentation clarity
- example quality
- replay and logging ergonomics
- schema export or validation tooling
- lightweight tests for the public subset

## Please Avoid

- adding real or sensitive data
- adding unpublished prompts, evaluations, or internal research assets
- expanding the repository into a clinical product claim

## How To Contribute

1. Open an issue describing the change you have in mind.
2. Keep pull requests small and focused.
3. Prefer synthetic fixtures and reproducible examples.
4. Document any change that affects public scope or repository boundaries.
